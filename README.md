## 📌 Project Overview
Infrastructure planning analysis for Tata Power's EV charging network 
across 18 Bengaluru districts, covering 1,14,000 charging sessions, 
65 active stations, and 98 proposed expansion projects (Aug–Dec 2024).

## 🔍 Key Findings
| Metric | Value |
|--------|-------|
| Total Sessions | 1,14,000 |
| Total Revenue | ₹4,96,93,947 |
| Active Stations | 65 |
| Registered Customers | 8,000 |
| Total kWh Delivered | 28,59,772 |
| High Priority Projects | 54 of 98 |
| Subsidy Eligible Projects | 55 |
| Total Plugs | 390 |

## 💡 Business Insights
- Indiranagar leads all districts with 8,901 sessions — highest EV demand zone
- October was peak month (39,606 sessions) — November dropped sharply, needs investigation
- ChargeZone leads operator revenue (₹1.05Cr) — Tata Power ranks 4th among 5 competitors
- 49.9% of customers are Low-Mid income tier — pricing strategy should reflect this
- 54 of 98 proposed projects are High Priority — ready for immediate investment
- 23 projects reclassified from Medium to Low priority — negative ROI cannot be medium priority

## 🛠️ Tools & Techniques
- **Excel** — Multi-table data cleaning, dashboard design
- **Power Query** — 5-table data transformation pipeline
- **Power Pivot** — Relational data model
- **DAX Measures** — Custom KPI calculations
- **Slicers** — District, Priority Tier, Income Tier, Charger Type, Operator
- **Lollipop Chart** — Operator revenue comparison using scatter + error bars
- **Treemap-style visualization** — Charger type distribution
